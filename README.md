Code and data files used to create R Shinyapp projects.  

S&P 500 returns: Includes a line graph, bar graph, and summary statistics of S&P 500 Closing Values and multi-year returns from 1928-2023.
URL: https://christy-yau.shinyapps.io/SP500-data/

Harry Potter Text Analysis: Bar plots displaying data for a word trend analysis and sentiment score analysis for each chapter in each Harry Potter book. 
URL: https://christy-yau.shinyapps.io/hp_text_analysis/

CA Crash data: Bar plots displaying data on total crashes per month, top counties with most crashes, and weather vs collision severity. A map of each CA city showing the location of each crash. 
URL: https://christy-yau.shinyapps.io/crashes-ca/

Health Insurance Claim Denials: Bar plots displaying percentage of denied claims and resulting amount of out-of-pocket (OOP) costs by income and race/ethnicity. 
URL: https://christy-yau.shinyapps.io/denied_claims_oop/
